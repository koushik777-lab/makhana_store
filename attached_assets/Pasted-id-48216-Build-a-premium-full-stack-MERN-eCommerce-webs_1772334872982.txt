Build a premium full-stack MERN eCommerce website for a food brand that sells Makhanas (Fox Nuts).

The website should look modern, premium, funky, and interactive, similar to modern D2C brands.

Use the following technologies:

Frontend:
- React.js
- Tailwind CSS
- Framer Motion (for animations)
- Axios
- React Router

Backend:
- Node.js
- Express.js

Database:
- MongoDB with Mongoose

Authentication:
- OTP verification using mobile number

Payments:
- Razorpay payment gateway integration

The website must have a clean folder structure, reusable components, and production-level code.

AUTHENTICATION SYSTEM

Customer must login to place an order.

Create two pages:

REGISTER PAGE

Fields:

- Username
- Mobile number
- Password

After submitting:

Redirect to OTP verification page.

OTP verification:
User enters OTP sent to mobile number.

After verification:
Account created successfully.

Store user in MongoDB.

--------------------------------------

LOGIN PAGE

Login using:

- Username
- Password

Check with database.

If match:

Login successful.

Use JWT authentication.
