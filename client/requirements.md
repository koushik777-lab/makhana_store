## Packages
zustand | Lightweight state management for Cart and UI state
framer-motion | Smooth animations and micro-interactions for a premium feel
lucide-react | High-quality icons

## Notes
- Zustand is used for the Cart state to easily access it from anywhere (Navbar, Shop, Checkout).
- Auth states rely on the `/api/auth/me` endpoint. 
- API URLs are dynamically built using the provided `@shared/routes` schema.
